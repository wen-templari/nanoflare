Static assets attached to a Worker deployment

full-demo now serves a static site

HTML, CSS, JavaScript, and the logo are bundled with Vite and deployed as attached assets. The Worker still handles dynamic API routes and can fetch assets through the ASSETS binding.

This example expects a KV namespace bound as VISITS_KV in nanoflare.json. Create it first with nanoflare kv namespace create visits, then replace the placeholder namespace id before deploying.
